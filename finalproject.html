The Final Project

Björgvin Theodór Hilmarsson Hafþór Helgi Heiðarsson Hallgrímur Kjartansson

This project is a group project executed by Björgvin Theodór Hilmarsson, Hafþór Helgi Heiðarsson and Hallgrímur Kjartansson. We are all third year mechanical engineering students from the university of Iceland. In this assignment we have to meet two of these three requirements.

1.Use a CAD software to build a model you can machine with the shopbot.

2. Use a CAD software to build a multy dimensional negative of a negative, Mill the mold using relevant stock using rough cut + finish cut, Cast the mold with silicone.

3. Read the MSDS for the material you intend to use.Design and fabricate a model (30cm^2) and produce a fiber composite of it.
